TEXTUAL INFORMATION CONTAINED IN THE PODCAST, FOR USE IN CASE OF DISABILITIES

Hello, we are group C and we would like to welcome you to our website. This website was made as part of our CSE/CIT 280 Sophomore Project class, and the purpose of this podcast is to show you the basic structure and functionality of our website.

The website’s topic is nutrition. We chose a file from Factual.com that contained a table filled in with Trader Joe’s products and their nutrition facts. We then took that information and organized it in a more user-friendly way. For those of you who are not familiar with Trader Joe’s, or who want to learn more about it, there is a link above to the official website.

Now, you can go to the working page by using the link at the top and have some hands-on experience.

For our actual working page, we decided to create three steps that the user could easily follow. In step 1, we included two external links to a website containing some useful Calories calculators; it contains the values needed for a healthy lifestyle, the user can enter his/her specific information, and it will display the recommended daily intake of Calories based on each person's metabolism. It is a handy tool to be used as a guideline. Then, coming back to our working page, we can continue with step 2.

In step 2, we organized the information displayed in the Factual.com site using a drop-down box, in which the user can select only one value at a time, and it will display the most important nutrition facts for the product selected. Then, once clicking the “submit” button, part 3 will be visible.

In step 3, the user will be able to see a graphic representation, based on the percent daily values of the standard nutrition values from the product previously selected on the drop-down list. As the color code suggests, each horizontal bar from the graph is mapped to a specific value. The different bars are meant to serve as a comparison of the different values within each product.

We hope you really enjoy the website, and find it helpful and useful. Our purpose was to come up with a good solution of visualizing those products sold by Trader Joe’s and have a supplementary material that could be used as a quick way to search for a specific item, instead of having to go through a whole table of values. It could be used as a daily diet planner, as well as a handy material to use in case you want to buy some of those products.

Thanks for visiting our website! We would like to see you as a returning user.
